This asset includes a modular sci-fi environment set with floor tiles, walls, ledges,
ladders, buttons, cables, lights, doors, pillars and more.
All parts are designed to mix and match on a consistent grid so you can build
your own levels and spaces freely.
 
Also included dedicated animations for the Mecha_00 character https://jimmys7777.itch.io/mech-00 :
a button-press interaction and a modular ladder climb with root motion.
 
Unity Package included with example scene and premade prefabs for fast scene assembly.

---
Textures

This package requires URP (Universal Render Pipeline).
Materials use URP/Lit and URP/Unlit shaders.

If your project uses HDRP or Built-in pipeline, materials will appear pink.
To fix: select the material → Inspector (top) → change shader to HDRP/Lit or Standard.

MATERIALS
No shader graphs. Simple texture setup:
  • Albedo  → texture
  • Emissive → same texture
Easy to adjust and compatible with any pipeline after a quick shader swap.

All meshes and textures are modular and easy to mix and match.
Each mesh uses one or two material slots:
Color – for the main look
Outline – for an unlit outline color
In the folder TexturePalettes, you’ll find color pallets textures.
You can apply any of these color textures to any mesh Color Material.
All meshes share the same texture layout, so the palettes are interchangeable.

There are included animations to use with the Mecha_00 character
and are designed to interact directly with Env_00 props.
Animations run at 60 fps.
 
Button Press:
  Mech_PressButton  - interaction animation for use with Rig_Button_0 / Rig_Button_1.
 
Ladder Climb (root motion):
  Mech_Climb_Enter   - entry animation when grabbing the bottom part.
  Mech_Climb_Idle    - indle animation on the ladder.
  Mech_Climb_Climb   - looping climb cycle with root motion.
  Mech_Climb_Exit    - exit animation when reaching the top of a ladder.
